# Data Governance: API → OLTP → Data Lake

This project demonstrates an end-to-end data governance framework
covering API ingestion, OLTP storage, and raw data lake ingestion.

## Scope
- API data contracts
- Data ownership & stewardship
- PII classification
- Data quality rules
- Security & access governance

## Architecture
API → API Gateway → OLTP → Data Lake (Raw Layer)

## Tools (Student Version)
- GitHub for documentation
- Excel for metadata
- Markdown for policies
